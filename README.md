# EDU Media Systems

Marketing site for EDU Media Systems — studio-grade media systems for schools (The Podcart hardware, Offloadr software, producer workflow).

## Stack

- Vite + React 19 + TypeScript
- Tailwind CSS v4 (`@tailwindcss/vite`)
- Inter (Google Fonts)
- Deployed on Vercel

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/`. Vercel serves it as a static SPA via `vercel.json`.
